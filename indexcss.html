Stylesheets

Styles can be applied externally to the style attribute by specifying:

  • tagname
  • id
  • class

Styles can be defined in the style tag of the page or in a separate file and included using the link tag.

<link rel='stylesheet' type='text/css' href='theme.css'>

Styles by tagname

Includes the tag to apply the style to. Applies the style to EVERY tag in the page. Style definition may contain multiple styles.

Styles by id

Must include the id of the tag to style. Id value must start with a hash char (#). Applied to the tag (if any) with the id value.

Styles by class name

Class name can be any unique name. Name must be applied with a period (.) Applied to any tag using the classname in the class attribute.

Styles in a stylesheet

Styles can be defined in a file (.css) and include in a page using the <link> tag. Multiple stylesheeets can be included in the same page. Last style encountered wins!

<head> <link rel = "stylesheet" type="text/css" href="master.css"> <link rel="stylesheet" type="text/css" href="page.css">

If you use a style. Use it inline or in the style tag in the head. If you use a stylesheet, use the link tag.

Multiple applications of the same style

The same style can be applied to the same tag, id, or classname. Separate different tag, id, or classname by a comma before style definition

Selector tag inside tag

Selector specifics to style all tags that are contained inside another tag. Tags are separated by a space.
