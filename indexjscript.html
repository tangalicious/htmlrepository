Syntax

Javascript is case-sensitive. Statements should end with a semicolon. Javascript statements exist with <script> tags. *Pages should also include the lang="javascript"

In C#, the semicolon(;) is known as a "statement terminator".

In Javascript, the semicolon (;) is known as a "statement separater".

Types

Javascript has five basic types:

number var n = 1;
string var n = "1"
boolean var n = true;
array var n = [1, 2, 3];
object var n = {1};

Other types

null var n = null;
undefined var n = n;

Can use "typeof n" to get the type of a variable

Dynamically typed

All variables are defined using VAR. Type of variable is based on the data stored in the variable.

Dynamically type

Once initialized, a variable can be set with any other type

Arithmetic operations

  • Modulus(%): var z = x %y;//0
  • Increment(++): var z = x++;//z=4,x=5
  • Decrement(--): var z = --y;//z=1;y=1
  • var b = a++; //Increment before setting variable
  • var c = ++a;// Increment after setting variable

Strings

Collection of characters surrounded by quotes (single or double)

String functions

ToString() length indexOf() substr(..,..) toUpperCase()/toLowerCase()

Boolean

Possible values are true or false

Array

A collection of Javascript data of any type

Items are separated by commas and enclosed within square brackets e.g. [1,2,3] e.g. [1, "1", true, [1, 2, 3], 1.1"

Array functions

length, join(ch), push(x), pop()

null

absence of a value. Different from being undefined

Boolean Expression

Any logical expression that evaluates to a boolean value. Typically involves comparison and or logical operators. May use parentheses to altar precedence.

Statement block

A statement block is one or more statements surrounded by curly braces. Can be placed anywhere a single statement is defined.

Assignment

A single equal sign represents assignment

Evaluate everything on the right of the equal sign down to a single variable value. Place that value into the variable on the left of the equal sign.

Equality

Double equal: two objects are equal if they have the same or compatible value

if

Execute a statement block only if a boolean expression is true. Optional else caluse defines a statement block to be executed if boolean expression is false.

while

Execute a statement block repeatedly as long as the boolean expression is true.

for(indexed)

Loops through a set of statements modifying a variable after every iteration plus checks

foreach

Iterates through each time in a collection

Ternary operator

Used to assign one of the two values to a variable based on some boolean expression
